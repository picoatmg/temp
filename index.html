MUD Web Client Demo

This is the demo page for a simple MUD web client I am working on. It is written in HTML5 / Javascript using websockify by Joel Martin and some code by Yen of nightfall.org.

The client requires a recent version of a HTML5 browser and is currently only tested with Firefox and can only be used with the MUD MorgenGrauen (in German).

Note that this page is not part of the actual client. To start the actual client, click here: Start Demo
